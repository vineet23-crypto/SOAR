# AI-Based SOC and SOAR System

## Overview
This project demonstrates an AI-driven Security Operations Center (SOC) and Security Orchestration, Automation, and Response (SOAR) system using Splunk and Wazuh.

---

## Tools Used
- Wazuh (SOC - Threat Detection)
- Splunk (AI + SOAR - Analysis & Automation)

---

## Key Features
- Anomaly-based threat detection
- Brute force attack detection
- Automated incident response
- Real-time monitoring dashboards

---

## Architecture
Attacker → Wazuh → Splunk → Automation Script → Action

---

## Automation
- Detects suspicious activity
- Blocks malicious IP
- Sends alerts
